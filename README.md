# InternHub

InternHub is a full-stack internship finder platform built using Node.js and JavaScript.

## Features
- Search internships
- Post internship opportunities
- Responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express

## Future Improvements
- User authentication
- MongoDB database integration
- Internship filtering by category
- Company logo upload
